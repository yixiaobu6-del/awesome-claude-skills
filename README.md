Claude Code Skills 精选列表与发现工具。

## 功能

- **使用统计**：查看次数 × 1 + 复制 × 3 加权评分
- **TOP20 热门**：最受欢迎 Skill 排行
- **组合推荐**：10 个场景 × 5 个 Skill 组合
- **搜索筛选**：按分类/功能快速查找

## 使用

打开 `index.html` 即可使用。
