# Awesome Claude Skills

> Claude Code 技能模板库 - 20+ 精选技能模板，助你快速构建高效AI工作流

## 项目简介

本仓库收录了经过实战验证的 Claude Code 技能模板（Skills），分为三大类：**人物型**、**主题型**、**工具型**。每个模板都包含完整的 SKILL.md 结构、使用方法和示例场景，可直接复制使用或根据需求定制。

## 为什么需要 Skills？

Claude Code 的 Skills 机制让你可以：
- **预设角色**：让 Claude 扮演特定专家角色
- **固化流程**：将重复性工作流程标准化
- **提升效率**：减少每次重新描述需求的时间
- **保证质量**：通过结构化模板确保输出一致性

## 快速开始

### 1. 安装技能

将任意 `.md` 文件复制到你的 Claude Code Skills 目录：

```bash
# macOS/Linux
cp 人物型/费曼教学法.md ~/.claude/skills/

# Windows
copy 人物型\费曼教学法.md %USERPROFILE%\.claude\skills\
```

### 2. 使用技能

在 Claude Code 中输入斜杠命令：

```
/费曼教学法 请解释量子纠缠的概念
```

### 3. 自定义修改

每个技能模板都包含清晰的注释，你可以根据具体需求调整：

- 调整输出格式
- 增加约束条件
- 添加示例场景
- 修改角色设定

## 最佳实践

### 1. 技能命名规范
- 使用简洁明了的中文名称
- 体现核心功能或角色
- 避免使用特殊字符

### 2. 内容设计原则
- 单一职责：每个技能专注一个核心任务
- 清晰约束：明确边界和限制条件
- 示例丰富：提供典型使用场景

### 3. 迭代优化方法
- 记录使用反馈
- 分析失败案例
- 持续调整优化
