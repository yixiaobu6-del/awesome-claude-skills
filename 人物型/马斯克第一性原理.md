# 马斯克第一性原理

## 技能概述
模拟埃隆·马斯克推崇的第一性原理思维——回归物理学的本质，拆解问题到最基本的真理，然后从零开始重构解决方案。适用于需要突破性创新和颠覆性思考的场景。

## 使用场景
- 产品创新与技术突破
- 成本优化与效率提升
- 战略决策与商业模式设计
- 解决长期未解决的难题

## 使用方法

```
/马斯克第一性原理 <需要解决的问题或挑战>
```

例如：
```
/马斯克第一性原理 电动车电池为什么这么贵？
/马斯克第一性原理 如何降低软件开发的成本？
/马斯克第一性原理 教育为什么这么低效？
```

### 工作流程

1. **明确问题**：清晰定义要解决的问题
2. **拆解到基本元素**：剥离所有假设，找到不可再分的基本真理
3. **识别隐含假设**：找出行业默认但从未被质疑的假设
4. **从零重建**：基于基本真理重新设计方案
5. **路径对比**：比较新方案和传统路径的差异

## 示例输出

### 输入：电动车电池为什么这么贵？

**基本真理：**
- 电池的材料是锂、镍、钴、铝、碳等
- 这些材料在伦敦金属交易所有公开报价
- 电池包=电芯+结构件+热管理系统+BMS

**被质疑的假设：**
1. 假设一：电池包必须用大量结构件来保护电芯
2. 假设二：电芯必须做成圆柱形或方形
3. 假设三：电池包必须使用昂贵的冷却系统

**重构方案（马斯克的真实做法）：**
- 放弃"电池包保护电芯"的思维，改为"电芯自身承担结构功能"
- 采用CTC（Cell to Chassis）技术，电芯直接集成到底盘
- 大规模自建电池工厂，一体化压铸减少零部件数量
- 转向磷酸铁锂，降低对钴的依赖

**结果：** 电池成本从2010年的$1,100/kWh降至2023年的约$100/kWh。

## 组合使用

- **+ 费曼教学法**：先理解问题本质，再重构解决方案
- **+ 塔勒布反脆弱**：评估新方案的反脆弱性
- **+ 芒格多元思维**：从多个学科视角审视重构方案

## 注意事项

- 第一性原理不是否定所有经验，而是质疑未经验证的假设
- 纯粹的第一性原理可能忽略实际工程约束
- 适合突破性创新，但日常决策可能不需要如此深度
