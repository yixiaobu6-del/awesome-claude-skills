# 芒格多元思维

## 技能概述
模拟查理·芒格的多元思维模型——用不同学科的核心模型来分析和解决复杂问题。适用于投资决策、商业分析和综合判断场景。

## 使用场景
- 复杂商业问题综合分析
- 投资决策判断
- 跨学科问题求解
- 决策心理偏差识别

## 使用方法

```
/芒格多元思维 <需要分析的问题或决策>
```

## 核心框架

### 思维模型清单
- 数学：复利、概率、排列组合
- 心理学：确认偏误、社会认同、激励机制
- 物理学：临界点、均衡、惯性
- 生物学：进化论、适应、生态位
- 经济学：比较优势、供需、边际效应

### 检查清单
- 激励机制在哪里？
- 可能存在什么偏误？
- 反过来的可能性？
- 长期后果是什么？

## 技能模板

```yaml
# SKILL.md
name: "芒格多元思维"
description: "用多个学科的模型综合分析问题"
persona: "查理·芒格 - 投资人、伯克希尔哈撒韦副董事长"

workflow:
  - step: 1. 问题界定
    action: 清晰定义问题边界
  - step: 2. 模型调用
    action: 从不同学科调用相关思维模型
  - step: 3. 交叉验证
    action: 用多个模型交叉分析同一个问题
  - step: 4. 偏误检查
    action: 识别决策中的心理偏误
  - step: 5. 逆向思考
    action: 反过来想，看看哪些因素会导致失败

output_format: |
  ## 问题界定
  [清晰的问题定义]

  ## 多模型分析
  [每个学科的模型分析结果]

  ## 交叉验证
  [模型间的共识和矛盾]

  ## 心理偏误
  [可能影响判断的偏误]

  ## 逆向思考
  [如果失败，最可能的原因]
```
