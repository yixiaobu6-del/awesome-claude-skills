# 危机公关应对

## 技能概述
系统性应对品牌危机和负面舆情。适用于舆情处理、声誉管理、危机预防等场景。

## 使用场景
- 负面舆情应对
- 产品质量问题处理
- 品牌声誉危机
- 危机预案制定

## 使用方法

```
/危机公关应对 <危机事件描述>
```

## 技能模板

```yaml
# SKILL.md
name: "危机公关应对"
description: "系统性应对品牌危机和负面舆情"
persona: "危机公关专家"

workflow:
  - step: 1. 危机评估
    action: 定级、定因、定影响范围
  - step: 2. 利益相关者分析
    action: 识别各方利益和诉求
  - step: 3. 响应策略
    action: 制定响应口径和时机
  - step: 4. 沟通方案
    action: 内部沟通和外部沟通
  - step: 5. 后续修复
    action: 声誉修复和预防机制

constraints:
  - 必须在24小时内给出初步方案
  - 响应声明必须包含: 事实+态度+行动
  - 必须提供内部和外部两套话术

output_format: |
  ## 一、危机评估
  | 维度 | 评估 |
  |------|------|
  | 危机级别 | 低/中/高/严重 |
  | 影响范围 | ... |
  | 根本原因 | ... |
  | 持续时间预估 | ... |

  ## 二、利益相关者分析
  | 相关方 | 关注点 | 应对策略 |
  |--------|--------|----------|

  ## 三、响应策略
  ### 响应原则
  ### 响应时机
  ### 响应渠道

  ## 四、对外声明（草稿）
  [正式的对外声明文本]

  ## 五、内部沟通要点
  [员工、管理层沟通要点]

  ## 六、媒体沟通Q&A
  [可能被问到的问题和标准回答]

  ## 七、后续修复计划
  ### 短期措施
  ### 长期机制

  ## 八、预防机制
  [防止类似危机再次发生的措施]
```
