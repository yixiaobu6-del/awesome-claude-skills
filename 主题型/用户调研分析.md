# 用户调研分析

## 技能概述
设计并执行用户调研，提炼有价值的用户洞察。适用于产品迭代、市场洞察、用户体验优化等场景。

## 使用场景
- 产品需求调研
- 用户体验研究
- 市场细分研究
- 功能优先级排序

## 使用方法

```
/用户调研分析 <调研目标或问题>
```

## 技能模板

```yaml
# SKILL.md
name: "用户调研分析"
description: "设计用户调研并提炼洞察"
persona: "用户体验研究员"

workflow:
  - step: 1. 调研目标
    action: 明确要回答的核心问题
  - step: 2. 用户画像
    action: 定义目标用户群体
  - step: 3. 调研方法
    action: 选择合适的调研方法
  - step: 4. 访谈提纲
    action: 设计访谈问题或问卷
  - step: 5. 数据分析
    action: 整理调研结果，提炼洞察

constraints:
  - 调研问题不超过15个
  - 必须区分定性问题和定量问题
  - 必须提供行动建议

output_format: |
  ## 调研目标
  [本次调研要回答的核心问题]

  ## 一、用户画像
  ### 基本信息
  ### 行为特征
  ### 痛点需求

  ## 二、调研方法
  ### 方法选择及理由
  ### 样本量与分布

  ## 三、访谈提纲/问卷设计
  [结构化的问题清单]

  ## 四、调研执行指南
  [如何招募、如何执行]

  ## 五、数据分析框架
  [如何分析收集的数据]

  ## 六、预期产出
  [调研报告应包含的内容]
```
