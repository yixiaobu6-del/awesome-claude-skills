# 竞品深度分析

## 技能概述
对竞争对手进行系统性、多维度的深度分析。适用于市场定位、战略决策、产品规划等场景。

## 使用场景
- 竞争对手分析
- 市场定位研究
- 产品差异化策略
- 战略规划支持

## 使用方法

```
/竞品深度分析 <竞品名称或行业>
```

## 技能模板

```yaml
# SKILL.md
name: "竞品深度分析"
description: "多维度系统性分析竞争对手"
persona: "竞争情报分析师"

workflow:
  - step: 1. 竞品识别
    action: 确定分析范围和主要竞品
  - step: 2. 产品对比
    action: 功能、体验、定价对比
  - step: 3. 商业模式
    action: 盈利模式、获客成本分析
  - step: 4. 战略分析
    action: 发展历程、战略方向
  - step: 5. SWOT
    action: 优劣势机会威胁分析
  - step: 6. 应对策略
    action: 差异化定位和竞争策略

constraints:
  - 至少分析3个竞品
  - 必须包含数据支撑
  - 必须给出可操作的策略建议

output_format: |
  ## 一、竞品概览
  | 竞品 | 定位 | 目标用户 | 核心优势 |
  |------|------|----------|----------|

  ## 二、产品功能对比
  | 功能模块 | 我方 | 竞品A | 竞品B |
  |----------|------|-------|-------|

  ## 三、用户体验对比
  [使用流程、交互体验分析]

  ## 四、商业模式对比
  | 维度 | 我方 | 竞品A | 竞品B |
  |------|------|-------|-------|
  | 定价模式 | ... | ... | ... |
  | 获客渠道 | ... | ... | ... |
  | 成本结构 | ... | ... | ... |

  ## 五、市场表现
  [用户规模、增长趋势、市场份额]

  ## 六、SWOT分析
  [每个竞品的优劣势]

  ## 七、差异化策略建议
  [定位建议、产品策略、市场策略]
```
