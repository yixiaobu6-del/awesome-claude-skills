# 深度报告生成

## 技能概述
生成结构完整、逻辑严密的深度研究报告。适用于行业研究、战略分析、市场洞察等需要系统性输出的场景。

## 使用场景
- 行业研究报告
- 战略分析报告
- 市场调研报告
- 竞争格局分析

## 使用方法

```
/深度报告生成 <报告主题>
```

## 技能模板

```yaml
# SKILL.md
name: "深度报告生成"
description: "生成结构完整的深度研究报告"
persona: "资深行业分析师"

workflow:
  - step: 1. 背景梳理
    action: 行业背景、政策环境、发展阶段
  - step: 2. 市场规模
    action: TAM/SAM/SOM分析，增长预测
  - step: 3. 竞争格局
    action: 主要玩家、市场份额、竞争态势
  - step: 4. 趋势分析
    action: 技术趋势、消费趋势、政策趋势
  - step: 5. 投资建议
    action: 机会点、风险点、建议关注方向

constraints:
  - 必须包含数据支撑
  - 必须引用信息来源
  - 每个章节必须有小结

output_format: |
  ## 执行摘要
  [报告核心结论]

  ## 一、行业背景
  ### 1.1 行业定义与分类
  ### 1.2 发展历程
  ### 1.3 政策环境

  ## 二、市场规模分析
  ### 2.1 市场规模测算
  ### 2.2 细分市场结构
  ### 2.3 增长驱动因素

  ## 三、竞争格局
  ### 3.1 主要参与者
  ### 3.2 市场份额
  ### 3.3 竞争策略分析

  ## 四、发展趋势
  ### 4.1 技术演进方向
  ### 4.2 消费趋势变化
  ### 4.3 行业预测

  ## 五、投资建议
  ### 5.1 机会分析
  ### 5.2 风险提示
  ### 5.3 建议关注

  ## 附录：数据来源与参考
```
