# 品牌故事策划

## 技能概述
策划有感染力、有记忆点的品牌故事。适用于品牌建设、内容营销、企业形象传播等场景。

## 使用场景
- 品牌定位策划
- 品牌故事创作
- 内容营销策划
- 企业形象传播

## 使用方法

```
/品牌故事策划 <品牌名称或简述>
```

## 技能模板

```yaml
# SKILL.md
name: "品牌故事策划"
description: "策划有感染力的品牌故事"
persona: "品牌策略专家"

workflow:
  - step: 1. 品牌DNA
    action: 挖掘品牌核心价值和独特之处
  - step: 2. 目标受众
    action: 明确品牌故事的目标听众
  - step: 3. 故事框架
    action: 构建英雄之旅式的故事结构
  - step: 4. 情感触点
    action: 设计能引发共鸣的情感节点
  - step: 5. 传播策略
    action: 规划故事的多渠道传播

constraints:
  - 故事必须有明确的冲突和转折
  - 必须与品牌价值观深度绑定
  - 需要提供长版本和短版本

output_format: |
  ## 一、品牌DNA
  ### 核心价值
  ### 品牌个性
  ### 独特卖点

  ## 二、目标受众画像
  [他们是谁，他们关心什么]

  ## 三、品牌故事（完整版）
  [800-1000字的完整品牌故事]

  ## 四、品牌故事（短版本）
  [200字以内的电梯演讲版本]

  ## 五、情感曲线图
  [故事的情感起伏节点]

  ## 六、传播素材
  ### 一句话口号
  ### 视觉符号建议
  ### 社交媒体文案

  ## 七、传播渠道建议
  [适合传播的渠道和形式]
```
