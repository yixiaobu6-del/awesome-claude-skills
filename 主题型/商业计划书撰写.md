# 商业计划书撰写

## 技能概述
撰写专业、规范、有说服力的商业计划书。适用于融资路演、项目立项、战略提案等场景。

## 使用场景
- 创业融资路演
- 新项目立项申请
- 战略投资提案
- 业务转型规划

## 使用方法

```
/商业计划书撰写 <项目名称或简述>
```

## 技能模板

```yaml
# SKILL.md
name: "商业计划书撰写"
description: "撰写专业规范的商业计划书"
persona: "投资背景的商业计划书专家"

workflow:
  - step: 1. 执行摘要
    action: 项目亮点、融资需求、核心价值
  - step: 2. 市场机会
    action: 痛点分析、市场规模、时机判断
  - step: 3. 产品方案
    action: 产品定位、核心功能、技术壁垒
  - step: 4. 商业模式
    action: 盈利模式、获客方式、增长策略
  - step: 5. 团队介绍
    action: 核心成员、能力互补、股权结构
  - step: 6. 财务规划
    action: 营收预测、成本结构、融资使用

constraints:
  - 执行摘要不超过2页
  - 必须有3-5年财务预测
  - 必须说明融资用途
  - 风险分析不可回避

output_format: |
  ## 执行摘要
  [一句话定位][核心亮点][融资需求]

  ## 一、公司概述
  ### 1.1 公司简介
  ### 1.2 使命愿景
  ### 1.3 发展里程碑

  ## 二、市场机会
  ### 2.1 痛点分析
  ### 2.2 市场规模
  ### 2.3 时机分析

  ## 三、产品与服务
  ### 3.1 产品定位
  ### 3.2 核心功能
  ### 3.3 技术壁垒
  ### 3.4 产品路线图

  ## 四、商业模式
  ### 4.1 盈利模式
  ### 4.2 获客策略
  ### 4.3 增长引擎

  ## 五、竞争分析
  ### 5.1 竞品对比
  ### 5.2 差异化优势
  ### 5.3 护城河构建

  ## 六、团队介绍
  ### 6.1 核心团队
  ### 6.2 顾问团队
  ### 6.3 股权结构

  ## 七、财务规划
  ### 7.1 营收预测
  ### 7.2 成本结构
  ### 7.3 资金使用计划

  ## 八、风险与对策
  ### 8.1 主要风险
  ### 8.2 应对措施

  ## 附录
```
