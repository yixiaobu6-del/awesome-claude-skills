安装指引

1查看模板详情

点击任意技能卡片，查看完整的 SKILL.md 模板代码。

2复制技能内容

在详情弹窗中点击「复制技能内容」按钮，将完整的 SKILL.md 代码复制到剪贴板。

3保存到项目

在项目根目录创建 .claude/skills/ 文件夹，将复制的内容保存为 SKILL.md 文件。

4全局安装（可选）

如需全局可用，保存到 ~/.claude/skills/ 目录，或在 claude.json 中配置 "skills" 路径。

5在 Claude Code 中使用

安装后重启 Claude Code，在对话中输入 /技能名称 即可调用对应技能。
