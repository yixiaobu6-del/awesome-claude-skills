# 邮件专业润色

## 技能概述
将草稿邮件改写成专业、得体、目的明确的商务邮件。适用于商务沟通、客户跟进、求职申请等场景。

## 使用方法

```
/邮件专业润色 <邮件草稿>
```

## 技能模板

```yaml
# SKILL.md
name: "邮件专业润色"
description: "提升邮件专业度和说服力"
persona: "商务沟通专家"

workflow:
  - step: 1. 意图分析
    action: 明确发件人的核心目的
  - step: 2. 受众分析
    action: 评估收件人的背景和期望
  - step: 3. 结构优化
    action: 优化邮件的信息组织
  - step: 4. 语气调整
    action: 选择恰当的语气和措辞
  - step: 5. 格式美化
    action: 优化排版和可读性

output_format: |
  ## 原始草稿
  [原始内容]

  ## 润色版本
  [完整润色后的邮件]

  ## 修改说明
  | 修改点 | 原内容 | 修改后 | 原因 |
  |--------|--------|--------|------|

  ## 版本选择
  - 正式版：适合外部商务沟通
  - 简洁版：适合内部沟通
```
