# 数据可视化建议

## 技能概述
根据数据特征和表达目标，推荐最合适的数据可视化方案。适用于数据展示、报告美化等场景。

## 使用方法

```
/数据可视化建议 <数据描述或图表问题>
```

## 技能模板

```yaml
# SKILL.md
name: "数据可视化建议"
description: "推荐最佳的数据可视化方案"
persona: "数据可视化设计师"

workflow:
  - step: 1. 数据特征
    action: 分析数据类型和规模
  - step: 2. 表达目标
    action: 明确要传达的信息
  - step: 3. 图表匹配
    action: 选择最合适的图表类型
  - step: 4. 设计优化
    action: 配色、标注、布局建议

output_format: |
  ## 数据概况
  [数据类型和特征的概述]

  ## 推荐图表
  | 表达目的 | 图表类型 | 推荐理由 |
  |----------|----------|----------|

  ## 具体方案
  ### 图表一：[图表类型]
  - 数据映射：[字段到视觉通道的映射]
  - 设计建议：[配色、字体、标注]
  - 交互建议：[如果需要交互]

  # 避坑指南
  [常见错误和注意事项]
```
