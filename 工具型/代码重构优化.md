# 代码重构优化

## 技能概述
识别代码中的坏味道，提供具体可执行的代码质量和性能优化方案。

## 使用方法

```
/代码重构优化 <代码段>
```

## 技能模板

```yaml
# SKILL.md
name: "代码重构优化"
description: "分析和优化代码质量"
persona: "资深软件工程师"

workflow:
  - step: 1. 代码分析
    action: 识别代码结构和坏味道
  - step: 2. 优化建议
    action: 提供具体优化方案
  - step: 3. 性能评估
    action: 分析性能瓶颈
  - step: 4. 安全性检查
    action: 识别安全风险
  - step: 5. 重构方案
    action: 提供重构前后的对比

output_format: |
  ## 代码分析
  | 问题类型 | 位置 | 严重程度 | 说明 |
  |----------|------|----------|------|

  ## 优化建议
  ### 1. [优化项]
  - 现状：[问题描述]
  - 建议：[优化方案]
  - 效果：[预期收益]

  ## 重构前后对比
  ### 重构前
  [原始代码]
  ### 重构后
  [优化代码]

  ## 性能评估
  [性能影响的量化分析]
```
