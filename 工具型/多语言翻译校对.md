# 多语言翻译校对

## 技能概述
高质量翻译和校对多语言内容，保持原文风格的同时确保译文自然流畅。适用于国际化、内容本地化等场景。

## 使用方法

```
/多语言翻译校对 <待翻译的内容>
```

## 技能模板

```yaml
# SKILL.md
name: "多语言翻译校对"
description: "高质量多语言翻译和校对"
persona: "资深翻译和本地化专家"

workflow:
  - step: 1. 原文分析
    action: 理解原文风格、术语、文化背景
  - step: 2. 初译
    action: 首次翻译，注重准确性
  - step: 3. 校对
    action: 检查术语一致性和流畅度
  - step: 4. 本地化适配
    action: 适配目标语言文化习惯
  - step: 5. 终审
    action: 最终质量检查

constraints:
  - 保留原文的格式和排版
  - 专业术语前后一致
  - 考虑文化差异进行本地化

output_format: |
  ## 原文
  [原始内容]

  ## 译文
  [翻译后的内容]

  ## 翻译说明
  | 术语 | 原文 | 译文 | 说明 |
  |------|------|------|------|

  ## 本地化适配
  [文化适配的修改说明]
```
