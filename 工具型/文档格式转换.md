# 文档格式转换

## 技能概述
在不同文档格式之间进行高质量转换，保持结构和样式完整。适用于内容迁移、文档标准化等场景。

## 使用方法

```
/文档格式转换 <需要转换的内容>
```

## 技能模板

```yaml
# SKILL.md
name: "文档格式转换"
description: "高质量文档格式互转"
persona: "文档工程师"

workflow:
  - step: 1. 源格式分析
    action: 识别源文件的格式和结构
  - step: 2. 目标规划
    action: 确定目标格式和样式
  - step: 3. 转换执行
    action: 保持内容和结构完整
  - step: 4. 兼容性检查
    action: 检查目标格式的兼容性

output_format: |
  ## 转换说明
  [源格式->目标格式]

  ## 转换结果
  [转换后的完整内容]

  ## 结构映射
  | 源结构 | 目标结构 | 样式说明 |
  |--------|----------|----------|

  ## 注意事项
  [格式差异和处理说明]
```
