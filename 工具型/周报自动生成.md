# 周报自动生成

## 技能概述
根据工作日志或口头描述，自动生成结构清晰、重点突出的周报。适用于日常汇报、进度同步等场景。

## 使用方法

```
/周报自动生成 <本周的工作描述>
```

## 技能模板

```yaml
# SKILL.md
name: "周报自动生成"
description: "自动生成结构化周报"
persona: "效率专家"

workflow:
  - step: 1. 信息收集
    action: 整理本周工作内容
  - step: 2. 分类归纳
    action: 按类型/项目分组
  - step: 3. 重点提炼
    action: 标注关键成果
  - step: 4. 数据补充
    action: 补充量化数据
  - step: 5. 下周规划
    action: 制定下周计划

output_format: |
  ## 本周工作总结

  ### 重点成果
  [已完成的核心KPI]

  ### 工作详情
  | 项目 | 任务 | 状态 | 备注 |
  |------|------|------|------|

  ### 关键数据
  [量化指标对比]

  ## 问题与风险
  [当前问题和对策]

  ## 下周计划
  [优先级排序的任务]
```
