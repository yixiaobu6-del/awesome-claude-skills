# 会议纪要生成

## 技能概述
将会议录音文字或笔记整理为结构化的会议纪要。适用于项目管理、任务追踪、团队协作等场景。

## 使用方法

```
/会议纪要生成 <会议内容记录>
```

## 技能模板

```yaml
# SKILL.md
name: "会议纪要生成"
description: "结构化整理会议记录"
persona: "项目经理"

workflow:
  - step: 1. 内容提取
    action: 提取关键讨论点
  - step: 2. 决策记录
    action: 明确会议达成的决策
  - step: 3. 任务分配
    action: 输出行动项和负责人
  - step: 4. 定期跟踪
    action: 设置后续跟踪节点

output_format: |
  ## 会议基本信息
  - 时间：
  - 主题：
  - 参会人：
  - 记录人：

  ## 会议目标
  [本次会议要达成的目标]

  ## 讨论要点
  ### 1. [议题一]
  [讨论内容要点]

  ### 2. [议题二]

  ## 决策记录
  | 决策 | 依据 | 影响 |
  |------|------|------|

  ## 行动项
  | 任务 | 负责人 | 截止日期 |
  |------|--------|----------|

  ## 下次会议
  - 时间：
  - 议题：
```
